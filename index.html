About AstroCalc

AstroCalc is a comprehensive astronomical calculation toolkit that allows you to explore the mechanics of our universe. From orbital dynamics to black hole physics and relativistic effects, this application provides accurate calculations to help understand astronomical phenomena.

Features:
  • Orbital mechanics (speed, period, escape velocity)
  • Black hole physics (Schwarzschild radius, photon sphere)
  • Gravitational calculations
  • Relativistic effects
  • Solar system data browser

This static web version runs entirely in your browser. All calculations are performed locally using JavaScript implementations of astrophysical formulas.
